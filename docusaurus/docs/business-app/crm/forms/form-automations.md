You can create and manage workflows directly from a form using the `Workflow` tab.

## Set up a form workflow

1. Go to **CRM** > **Forms** and open an existing form (or create a new one).
2. Click the **Workflow** tab in the form editor.
3. If no workflow exists, choose a template (e.g. "Send follow-up email when a form is submitted") or start from Blank.
4. Configure the suggested step (for example, select a campaign for the follow-up email) and click **Save**.
5. Toggle the workflow **On** using the switch in the top-right.

:::tip
Starting from a template is the fastest way to launch a reliable follow-up. You can always switch to **Advanced mode** to add steps later.
:::

## Build more complex workflows

- Click **Advanced mode** to open the full workflow builder.
- Add **Conditions** and additional **Actions** (for example, apply a tag, assign a task, send SMS).

:::note
The **Workflow** tab shows all workflows connected to the selected form.
:::

See [Workflows](../../automations/index) for the full workflow builder and more triggers and actions.
