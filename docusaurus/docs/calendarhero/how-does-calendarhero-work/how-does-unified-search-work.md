---
title: How does Unified Search work?
sidebar_position: 18
---

Quickly find any document before your next meeting with one search, whether it’s in Google Drive, Dropbox, or elsewhere. By using CalendarHero's unified search skill, you can easily sift through all the information across your connected data accounts. 

Once you have added integrations, you can search via web or chat. Scroll down to learn more. 

Search in Chat

In your connected chat platform (such as Slack or MSTeams), start a search by asking your assistant questions such as:

- *Where is the NDA file from last month?*

- *Search the company logos*

- *Where are the headshots*

Be patient as your assistant searches for you! Your assistant will return the results directly in chat. Click on a search result to be taken directly to the associated document; it opens in the 3rd party software where it lives - so you can be taken directly to the file you need! 


Search via Web

From the powerful CalendarHero web app go to the [Unified Search web interface](https://app.calendarhero.com/tasks/new/search) and enter a search term into the box provided. Be patient as CalendarHero searches across all of your search integrations to find associated information. 

  
Click on a search result to be taken directly to the associated document; it opens in the 3rd party software where it lives - so you can be taken directly to the file you need! 


####  

Adding Search Integrations

It's easy to add new integrations for Unified Search. Go to the CalendarHero Integration Directory ([Search tab](https://app.calendarhero.com/settings/accounts/add#search)) and simply click "Add" on the Integration you want to add. Once you have authorized the integration it will appear in your [Integrations list](https://app.calendarhero.com/settings/accounts/list).

  
If you are an Admin in a [Team Plan Account,](https://calendarhero.com/pricing) then you can also add integrations on behalf of your entire organization (from here: [https://app.calendarhero.com/org/accounts/add#search](https://app.calendarhero.com/org/accounts/add#search))
