---
title: How does Unified Search work?
sidebar_position: 18
---

Quickly find any document before your next meeting with one search, whether it’s in Google Drive, Dropbox, or elsewhere. By using CalendarHero's unified search skill, you can easily sift through all the information across your connected data accounts. 
 

Once you have added integrations, you can search via web or chat. Scroll down to learn more. 
 

**Search in Chat**
 

In your connected chat platform (such as Slack or MSTeams), start a search by asking your assistant questions such as:
 
 
- ***Where is the **NDA file from last month?* 
- ***Search the **company logos* 
- ***Where are the **headshots* 
 

Be patient as your assistant searches for you! Your assistant will return the results directly in chat. Click on a search result to be taken directly to the associated document; it opens in the 3rd party software where it lives - so you can be taken directly to the file you need!
