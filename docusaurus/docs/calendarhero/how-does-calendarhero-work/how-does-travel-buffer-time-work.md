---
title: How does Travel Buffer Time work?
sidebar_position: 24
---

When booking out-of-office meetings, it can be a challenge to remember to include travel time as a consideration. CalendarHero can automatically include travel buffer time for external meetings booked [via a chat platform](https://support.calendarhero.com/hc/en-us/articles/14078231086487-How-to-Schedule-a-Meeting-via-Chat) like Slack or MS Teams. 
 

If the travel location requires travel buffer time then your CalendarHero assistant will ask the meeting creator if they want to schedule travel time for the meeting.
