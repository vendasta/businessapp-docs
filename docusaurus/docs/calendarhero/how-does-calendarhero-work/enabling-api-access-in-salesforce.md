API access must first be enabled on your Salesforce account for CalendarHero to integrate. Your Salesforce Administrator controls your Profile and Permission Sets.
 

 

#### There are two ways to enable API access in Salesforce:
 
 
-  

Profile
  
-  

Permission Set
  
 

### Enable API access in Salesforce by Profile
 

1. Click the Gear icon and click **Setup.**
