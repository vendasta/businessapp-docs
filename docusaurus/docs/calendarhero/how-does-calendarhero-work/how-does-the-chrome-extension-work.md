The CalendarHero Chrome extension is no longer available and has been removed from the Chrome store.

However, we have some great alternatives that will help to supercharge your scheduling needs! Instead of using the Chrome extension, we recommend the following: 

CalendarHero for Gmail: Our Gmail add-on gives you shortcuts to our most popular features directly inside your inbox. Learn More

The Web Scheduler: CalendarHero’s web-based scheduler offers quick scheduling options and a hub for all your meeting tasks. Learn More

CalendarHero for Outlook: Like the Gmail add-on, Outlook users can access shortcuts to powerful scheduling features from their inboxes. Learn More
