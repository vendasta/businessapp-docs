---
title: CalendarHero Primary Booking Link
sidebar_position: 0
---

CalendarHero provides all users with a primary, default booking link. As shown below, this link cannot be deleted from CalendarHero as it ties back to the account. If users do not want to use this link, they can ignore it, and pin other meeting types to the dashboard so that those links appear first.



All other meeting links that are created by users can be deleted by clicking edit > delete. 



To pin other links to the dashboard, click the three dots next to the meeting link, and select 'pin to dashboard.' This will move the link to the top.
