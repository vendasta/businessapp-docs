---
title: Is it possible to be cc'd on my meeting request emails?
sidebar_position: 3
---

CalendarHero users can be cc'd on meeting request emails sent to their invitees. Click [here](/calendarhero/how-does-calendarhero-work/scheduling-page-what-do-invitees-see-when-i-invite-them-to-a-meeting) to learn more about what invitees see when you invite them.

#### Settings: Cc Notification

This notification setting can be managed from Meeting General Settings    
[https://app.calendarhero.com/settings/meeting#general](https://app.calendarhero.com/settings/meeting#general)

To enable check this setting: "Cc me when my CalendarHero assistant sends a meeting request".

This option is ON by default but can be disabled if required.


---

#### How does it work?  
*Email: *When enabled we Cc the meeting creator on the meeting invite request (e.g. the email that is sent to the invite recipient asking if they are available to meet). For example:

Jill invites Aseel to a meeting using CalendarHero (and allows her to choose a time)

Aseel receives the current CalendarHero invite email. If Cc is ON then Jill is also cc’d on the *same* invite email.
