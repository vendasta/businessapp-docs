title: How do I add a Room Booking provider?
sidebar_position: 22
---

CalendarHero’s Room Booking feature quickly integrates with MS Rooms, Google, Zoom Rooms, or [Robin](https://robinpowered.com/) to allow users to quickly and efficiently book a meeting room based on capacity.   
  
Learn how to set up your provider so you or your team can use the room booking feature.  
  
Want to learn how to use the Room Booking once a provider is added? Go here: [How does Room Booking Work?](https://support.calendarhero.com/hc/en-us/articles/14078963571863-How-does-Room-Booking-work-)
 

>  **Please note that this feature is only available for Team Plan users.** Before users can request a room a Team Plan Administrator must add a supported room provider.
