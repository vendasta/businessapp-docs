Your CalendarHero assistant lives where you work - inside your favorite chat platform, alongside your inbox, or directly in your browser!** **Below is a list of the support platforms CalendarHero integrates with. Click a link to learn more about each.   
  
If you are using more than one platform you can select which one receives notifications, like morning briefings: [How to set a preferred platform.](https://support.calendarhero.com/hc/en-us/articles/14078766252183-Can-I-customize-where-I-receive-my-notifications-)  
  
  

### Web Interface
 

CalendarHero started as a chat-based platform but now works on the web - just like you do. 
 

With our web-based interface, you can schedule meetings, without needing to install anything. It is the most powerful and popular way to use CalendarHero.
 
 
- Ready to get started? [Sign up here](https://calendarhero.com/start/) and then start scheduling meetings 
- Learn more about [scheduling a meeting on the web](https://support.calendarhero.com/hc/en-us/articles/14081616167191-How-do-I-Schedule-Meetings-)

Chat Platforms
 

CalendarHero supports all of your favorite chat platforms! Once installed you can chat with your Meetings Assistant using natural language. 
 
 
- Microsoft Teams - [Learn More](https://support.calendarhero.com/hc/en-us/articles/14078090316823-Using-CalendarHero-with-Microsoft-Teams)| [Install Teams](https://calendarhero.com/microsoft-teams/)  
- [Slack](https://calendarhero.com/slack) 
- [Google Hangouts Chat](https://calendarhero.com/hangouts) 
- [Skype](https://calendarhero.com/skype) 
- [Facebook Messenger](https://calendarhero.com/facebook-messenger) 
- [Webex Messaging](https://calendarhero.com/webex-meetings) 
- [SMS](https://calendarhero.com/sms)
