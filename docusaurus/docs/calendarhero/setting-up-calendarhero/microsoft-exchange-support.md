CalendarHero supports Microsoft's Exchange server for calendar synchronization, event creation, and contact synchronization:

[Online/Hybrid](#online) (2016+) with extended authentication and optional custom-hosted domain 

[On-Prem](#onprem) (2013+) that is hosted behind your firewall, with basic authentication (username, password, domain/forest, EWS server)

If your organization uses Microsoft Office 365, please sign in/register with the "Sign-in with Microsoft" button found on [https://calendarhero.com/start](https://calendarhero.com/start)

Microsoft Exchange Online

New User Registration:

1. Go to [https://app.calendarhero.com/register](https://app.calendarhero.com/register)

2. Select Exchange Online

3. Enter your email address at work

4. Click on the "Sign-Up" button

5. Authorize CalendarHero to access your account on the Microsoft authorization page


User Sign-In

1. Go to [https://app.calendarhero.com/login](https://app.calendarhero.com/login)

2. Select Exchange Online

3. Enter your email address at work

4. Click on the "Sign-In" button


Adding Exchange to an Existing CalendarHero User Account

If your user already has a CalendarHero account using another calendar such as Google G-Suite or Microsoft Office 365.

1. Go to [https://app.calendarhero.com/settings/accounts/add](https://app.calendarhero.com/settings/accounts/add)

2. Select Microsoft Exchange Online

3. Click on the Add button

4. Enter your email address

5. Click on the Add button

6. Accept the Microsoft authorization

Microsoft Exchange On-Prem

Special Requirements for Microsoft Exchange On-Prem users:

Your IT department will need to allow CalendarHero to access your user accounts before you can register. Enable users' accounts to be accessed externally which means that those accounts need permissions for external authentication, API read/write access to calendar events, and API read access to personal contacts. 

New User Registration

1. Go to [https://app.calendarhero.com/register](https://app.calendarhero.com/register)

2. Select Exchange On-Prem

3. Enter your email address at work, username, password, Exchange server, and optionally a domain/forest

4. Click on the "Sign-Up" button  


User Sign-In

1. Go to [https://app.calendarhero.com/login](https://app.calendarhero.com/login)

2. Select Exchange On-Prem

3. Enter your email address at work and your password

4. Click on the "Sign-In" button  


Adding Exchange to an Existing CalendarHero User Account

If your user already has a CalendarHero account using another calendar such as Google G-Suite or Microsoft Office 365.

1. Go to [https://app.calendarhero.com/settings/accounts/add](https://app.calendarhero.com/settings/accounts/add)

2. Select Microsoft Exchange On-Prem (click "Add")

3. Enter in your email address, username, password, Exchange server, and optionally domain/forest

4. Click on the Add button
