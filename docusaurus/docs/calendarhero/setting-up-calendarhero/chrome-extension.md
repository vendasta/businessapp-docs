**The CalendarHero Chrome extension is no longer available and has been removed from the Chrome store.**
 

We do, however, have some great alternatives that will help to supercharge your scheduling needs! Instead of using the Chrome extension, we recommend the following: 
 
 
-  **CalendarHero for Gmail: **Our [Gmail add-on](https://support.calendarhero.com/hc/en-us/articles/14078859754775-How-to-use-the-CalendarHero-Gmail-Add-on) gives you shortcuts to our most popular features directly inside your inbox. 
-  **The Web Scheduler: **CalendarHero’s [web-based scheduler](https://support.calendarhero.com/hc/en-us/articles/14081616167191-How-do-I-Schedule-Meetings-) offers quick scheduling options and a hub for all your meeting tasks.  
-  **CalendarHero for Outlook:** Like the Gmail add-on, [Outlook users](https://support.calendarhero.com/hc/en-us/articles/14078826956823-How-does-the-Microsoft-Outlook-Plugin-Work-) can access shortcuts to powerful scheduling features from their inboxes.
