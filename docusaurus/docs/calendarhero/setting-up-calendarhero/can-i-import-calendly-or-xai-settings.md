Moving over from another scheduling tool? You're not alone. 

To help you out CalendarHero makes it easy to import your *Calendly* 'Event Types' or your *x.ai* 'Meeting Templates', so you can quickly transfer your preferred settings into CalendarHero's powerful [Meeting Types](/calendarhero/scheduling-meetings/create-and-customize-a-meeting-type).  
  
Please note that you must be a CalendarHero Trial, Professional, or Team plan user in order to create new meeting types. [View Plans](https://calendarhero.com/pricing)

### Import x.ai

X.ai made the decision to sunset its standalone scheduling service [(read blog post)](https://blog.bizzabo.com/xai-sunset?utm_campaign=xai-sunset). To help transition users, the folks at x.ai built tools to help you export your x.ai data. Follow the instructions below to import this CSV into CalendarHero.

To import your x.ai meeting templates into CalendarHero meeting types, first, use the x.ai export link to download your x.ai CSV file. 

2. Next [login to CalendarHero](https://app.calendarhero.com/) and go to [Settings New Meeting Type](https://app.calendarhero.com/settings/meeting/new) to create a new meeting type. Select "Meeting" as the style. Note: Basic plan users will be prompted to upgrade to unlock Custom Meeting types.

3. From your new meeting type click the "Import X.ai (CSV)" button. You will be prompted to upload your CSV file from your computer. 


Select which x.ai meeting template you wish to import. The supported settings for that template will be imported into a new CalendarHero Meeting Type.

5. Once imported --- customize the additional CalendarHero meeting-type settings.

6. Click "Save" to save your new meeting type!

### Here are the x.ai settings CalendarHero can currently import:

Title

  - The title will be added as the CalendarHero [Default Meeting Title](/calendarhero/scheduling-meetings/how-do-i-add-or-customize-a-meeting-title)

Calendar Page Link 

  - Your link will be used to form the URL for your CalendarHero [Personal Scheduling Link](/calendarhero/scheduling-meetings/how-to-use-a-personal-scheduling-link).

  - Your link will also be used for the CalendarHero Meeting Type Name. The Name will be referenced when you want to apply the meeting-type settings to one of your meetings. *Update this as needed before you Save.*

Hours

  - Hours will be added to the CalendarHero Availability Windows (only supporting weekdays for now)

Breather Time 

  - Breather Time will be imported into CalendarHero [Buffer Times (After)](/calendarhero/what-can-calendarhero-do/can-i-add-buffer-time-to-my-meetings)

  - TIP! Buffer Time supports time both before AND after a meeting.

Lead Time

  - Lead Time is imported into CalendarHero [Lead Time](/calendarhero/scheduling-meetings/how-do-i-set-lead-time-minimum-notice)

Invitee Notes

  - This description copy will be added to CalendarHero [Meeting Instructions](/calendarhero/scheduling-meetings/how-to-add-meeting-instructions)

Duration

  - The event duration will be set as the default duration (e.g. how long will this meeting last? e.g. 1hr)

Location (Address)

  - An address in the location field will be added to the CalendarHero [location field](/calendarhero/scheduling-meetings/create-and-customize-a-meeting-type).

Location (Video Conferencing)  

  - For Video Providers we attempt to match keywords such as 'Zoom', 'video', and 'Skype' to one of your integrated default dynamic video providers. Please note you have to integrate your video provider first!

Guest Questions (Custom Fields)

  - Guest Questions will be added as CalendarHero [Invitee Questions](/calendarhero/what-can-calendarhero-do/how-do-i-ask-my-invitee-questions). Please note however we currently only support "textarea" import, which will import into CalendarHero 'Multi-line question.'

### Import Calendly

To import an existing Calendly event type, you will first need to copy your Calendly API key.  

1. Log in to your Calendly account at [https://calendly.com/login](https://calendly.com/login).

2. Go to the Calendly Integrations page at [https://calendly.com/integrations](https://calendly.com/integrations)

3. Copy your API key


Once you have copied your Calendly API key [login to CalendarHero](https://app.calendarhero.com/) and go to [Settings New Meeting Type](https://app.calendarhero.com/settings/meeting/new) to create a new meeting type. Select "Meeting" as the style.

1. From your new meeting type click the "Import Calendly" button.

2. Enter the API key and click "Go"

3. Select which Calendly event type you wish to import. The supported settings for that event type will be imported into a new CalendarHero [Meeting Type](/calendarhero/scheduling-meetings/create-and-customize-a-meeting-type).

4. Once imported --- customize the additional CalendarHero meeting-type settings.

5. Click "Save" to save your new meeting type!



### Here are the Calendly settings CalendarHero can currently import:

Name

  - A version of your event name is used to name your custom Meeting Type. We may simplify the name to a single word, and ensure it is unique. 

Description 

  - This event description copy will be added to the [Meeting Instructions](/calendarhero/scheduling-meetings/how-to-add-meeting-instructions)

Scheduling Link Code 

  - Your slug and username will be used to form the URL for your [Personal Scheduling Link](/calendarhero/scheduling-meetings/how-to-use-a-personal-scheduling-link).

Duration

  - The event duration will be set as the default duration (e.g. how long will this meeting last? e.g. 1hr)
