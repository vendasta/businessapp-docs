title: How to customize the Start Time display

Customize the display of available start times offered to your invitees per meeting type. Start time determines how often the time slots appear on the [Meeting Acceptance page](/calendarhero/how-does-calendarhero-work/scheduling-page-what-do-invitees-see-when-i-invite-them-to-a-meeting) (which is where your invitees select a time to meet with you). The example below shows 15-minute start time increments.


Start Time (Meeting Type)

By default, all CalendarHero users have a default 'Hourly Offset' setting in their [General Meeting Settings](/calendarhero/how-to/how-to-customize-general-meeting-settings). Start times however can now be customized* per *meeting type. To get started simply open your [meeting type](/calendarhero/scheduling-meetings/create-and-customize-a-meeting-type) and from the "Availability" tab customize the start time setting based on your needs.


Display Hourly Offsets: This is the legacy option. If this option is selected then the time slot displayed on your meeting acceptance page will reflect the "Hourly Offset" setting under your 'General Meeting Settings'. *(See the following section for more information).*

Display increments of: If this option is selected then the dropdown selection (e.g. 30min) will be used as the incremental value for start times on the meeting acceptance time slots. Dropdown options include 5 min, 10 min, 15 min, 20 min, 30 min, 45 min, 50 min, 60 min. 15 min is the default for all NEW meeting types   

  - Example 1: If “15 min” is selected for a 30-minute duration meeting, and availability starts at 9 a.m. - start times will appear as 9:00 a.m. - 9:30 a.m., 9:15 a.m. - 9:45 a.m., 9:45 a.m.- 10:15 a.m., 10:00 a.m. - 10:30 am, etc.

  - Example 2: If “10 min” is selected for a 30-minute duration meeting, and availability starts at 9 am - start times will appear as 9:00 a.m.-9:30 a.m., 9:10 a.m.-9:40 a.m., 9:20 a.m. - 9:50 a.m., 9:30 am -10:00 a.m., 9:40 a.m. - 10:10 a.m., 9:50 a.m. - 10:20 a.m., etc. 

Hourly Offsets (General Meeting Settings)


To view your default Hourly Offsets go to [General Settings](/calendarhero/how-to/how-to-customize-general-meeting-settings) Hourly Offsets ([https://app.calendarhero.com/settings/meeting/general](https://app.calendarhero.com/settings/meeting/general)).

These increment values are used for all meeting types where "Display Hourly Offset" is selected as the Start Time. Any meeting type that has "Display increments of x" enabled will override this setting.
