---
title: Can we remove the virtual assistant's messages from the CalendarHero dashboard permanently?
sidebar_position: 4
---

The first part of the message is hard-coded and unable to be customized by users; the second part of the message can be customized and it's optional.

  
"\{organizer\} would like to meet with you. Please select a time to meet by clicking the link below" This is a fixed component of any invite email from CH, and it can't be changed.

"Our designer has invited you to schedule a meeting ...." This message can be customized for every meeting type and is optional.
