---
title: How do I Schedule a Group Class?
sidebar_position: 13
---

Allow multiple attendees to easily self-book the same class or event. ***Group Class*** is recommended for group events or online training that requires individual sign-up. Each attendee can sign up at their convenience via a custom scheduling page (and [pay via credit card](https://support.calendarhero.com/hc/en-us/articles/14079113339671-How-to-Collect-Payments-for-your-meetings) if required). *Group Class* requires that the event date and time be pre-selected. 
 

If 'No Limit' is selected then the Group class supports up to 1000 attendees. Alternatively, you can choose to customize the attendee limit using [Max Attendees](https://support.calendarhero.com/hc/en-us/articles/14078270727959-How-do-I-set-a-Max-Attendee-limit-for-my-group-class-).
 

*Group Class *is now available for all users on the [Professional or Team Plan](https://calendarhero.com/pricing). 
 

**Please note:** Due to 3rd party calendar limitations *Group Class* is NOT compatible with the following booking calendars: *iCloud *and *Exchange On-Premises.* [Learn More](https://support.calendarhero.com/hc/en-us/articles/14077988850071-Calendar-not-supported-for-specific-settings).
