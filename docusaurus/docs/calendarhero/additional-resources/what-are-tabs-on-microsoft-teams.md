---
title: What are tabs on Microsoft Teams?
sidebar_position: 11
---

Microsoft users can access key CalendarHero settings inside Microsoft Teams by clicking on the integrated **CalendarHero Teams** tabs. Users may be prompted to first sign into CalendarHero with their linked Microsoft account. 
 

These tabs provide a view of the CalendarHero web app so that users can more quickly access key CalendarHero preferences inside MS Teams - specifically *Dashboard*, *Settings*, *Tasks,* and *Skills Help*. For full CalendarHero functionality, we recommend opening [https://app.calendarhero.com](https://app.calendarhero.com/) in your browser. 
 

Please note that due to integration limitations, these MS Teams tabs ( *Dashboard*, *Settings*, *Tasks,* and *Skills Help*) are only accessible to CalendarHero users who log in with Microsoft. If you cannot access these tabs (e.g. you use CalendarHero with Google) don't worry - you can continue to use the CalendarHero Automated Assistant via chat in MS Teams ([to make meeting requests in natural language](https://support.calendarhero.com/hc/en-us/articles/14081616167191-How-do-I-Schedule-Meetings-)), and simply access the additional CalendarHero functionality in any web browser using the full CalendarHero Web App ( [https://app.calendarhero.com](https://app.calendarhero.com)).
 

[Learn more about using MS Teams with CalendarHero here](https://support.calendarhero.com/hc/en-us/articles/14078090316823-Using-CalendarHero-with-Microsoft-Teams)
