# Yesware Overview

The Overview section covers Yesware setup, core features, and troubleshooting. New to Yesware? Start with [Getting Started with Yesware](../getting-started-with-yesware) to install and configure your account.

## What is Yesware?

Yesware is a sales engagement platform that integrates with your Gmail or Outlook inbox to help you:

- **Track emails** when recipients open and click your messages
- **Create multi-touch campaigns** to automate and personalize outreach
- **Schedule meetings** with Meeting Scheduler
- **Track presentations** to see engagement on shared documents
- **Sync with Salesforce** (Enterprise plans) for CRM integration

## Platform compatibility

Yesware supports **Gmail** (via Chrome extension) and **Outlook (Office 365)** (via O365 add-in). See [How do I install Yesware?](./installing-yesware/) for installation steps.

## Frequently asked questions

<details>
<summary>Can I use Yesware with a personal Gmail account?</summary>

Yes. Yesware works with both G Suite (Google Workspace) and personal Gmail accounts (`@gmail.com`).
</details>

<details>
<summary>Can I use Yesware with GoDaddy email addresses?</summary>

No. Yesware only supports Gmail and Outlook. The Outlook add-in requires a direct, Microsoft O365-hosted email address.
</details>

<details>
<summary>Can I use Yesware for more than one email address?</summary>

Yesware is licensed by email address. Each email needs its own license. You can add licenses at [app.yesware.com/account/manage](https://app.yesware.com/account/manage). Yesware does not work with Alias or "Send As" addresses.
</details>

<details>
<summary>Can I install Yesware on a shared mailbox?</summary>

No. Yesware works only with the primary email address linked to your account.
</details>
