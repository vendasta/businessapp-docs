# Can I use dynamic Templates with Salesforce data on my plan?

The feature is only available to [Enterprise plan](http://www.yesware.com/plans-and-pricing) holders who are [using Salesforce.](https://support.yesware.com/entries/24805668-How-do-I-enable-my-emails-replies-and-tracking-events-to-automatically-get-sent-to-Salesforce-)
For more information on how to use those dynamic fields, please [click here.](/docs/templates/using-templates/how-do-dynamic-templates-work-with-salesforce)
