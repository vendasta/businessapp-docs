# Spacing Issues in Templates

If you notice unexpected spacing in a template's email body, the most common cause is pasting content from external sources such as Microsoft Word, websites, or old emails.

When you paste from those sources, hidden formatting is carried over. It may not be visible in Gmail but can appear in other email clients.

## Fix

Paste your content into a plain-text editor first to strip hidden formatting, then copy it into the template editor.
