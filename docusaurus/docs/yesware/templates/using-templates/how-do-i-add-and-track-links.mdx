# Tracked Links in Templates

When you add hyperlinks to a template, they are automatically set up as tracked links so you can monitor engagement and prioritize follow-ups.

## Add a tracked link

1. Click the **chain link icon** in the template editor.
2. Enter the URL.
3. Click **OK**.

**Best practice:** Use descriptive hyperlink text (e.g., *Yesware website*) instead of raw URLs. This reduces the chance of your message being flagged as spam.

## Disable link tracking

Go to **Yesware Preferences** and toggle off link tracking for templates.
