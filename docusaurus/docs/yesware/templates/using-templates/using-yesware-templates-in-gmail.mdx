# Using Templates in Gmail

## How to insert a template

1. Open a new compose window and add the recipient's address to the **To** field.
2. Click the **Templates** button at the bottom of the compose window.
3. Browse **Recent**, **Favorites**, or your full library to find the template you want.
4. Hover over the template and choose:
   - **Fill In** — opens the template so you can complete any merge fields before inserting.
   - **Insert** — inserts the template as-is. Use this only for templates without merge fields.

## Favorite templates

Click the **star** next to any template name to favorite it. Favorites appear under the **Favorites** tab in the compose window for quick access.
