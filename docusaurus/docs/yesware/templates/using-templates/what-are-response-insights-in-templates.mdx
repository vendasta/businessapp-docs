# Response Insights

Response Insights provides data-backed suggestions to improve the predicted reply rates of your templates. Learn more about the algorithm on the [Yesware blog](https://www.yesware.com/blog/analyzed-data-sales-templates/).

## How to use it

1. In the template editor, click **Response Insights** in the lower right.
2. View your template grade and see which areas could be improved.
3. Click each category for detail on what makes an effective email.
