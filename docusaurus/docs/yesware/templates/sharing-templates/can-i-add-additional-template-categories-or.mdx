# Manage Template Folders

You can create folders to organize your templates.

## Create a folder

1. Open the **Templates library**.
2. Click the **folder +** icon in the upper right.
3. Name the folder and choose whether to save it to **Personal Templates** or a **Team**.
4. Click **Create**.

> Only Team Leaders can create team template folders.

## Edit or delete a folder

1. Click the folder in the Templates library.
2. Click the **pencil icon** to rename or the **trash can icon** to delete.
