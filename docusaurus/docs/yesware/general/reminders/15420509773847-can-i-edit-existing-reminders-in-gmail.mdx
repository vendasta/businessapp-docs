# Can I edit existing Reminders in Gmail?

**To edit an existing Reminder:**
- Open the email with the Reminder
- Click on the Reminder drop-down (the bell icon)
- This will show you the currently scheduled Reminder
- Click "Clear Reminder"
- A new Reminder setting will pop-up
- Set a new time and date, and save
