# Can I use Yesware with GoDaddy email addresses?

**No, Yesware only supports Gmail and Outlook at this time. **
While GoDaddy hosted emails can be used through Microsoft Exchange, the Yesware Outlook Add-in requires a direct, Microsoft O365-hosted email address.
