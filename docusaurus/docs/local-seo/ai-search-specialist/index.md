The AI Search Specialist is a conversational AI available with Local SEO Pro. It provides AI-powered insights about your local search performance and lets you run an AI Engine Optimization (AEO) audit of your website directly in the chat.

## Audit your website's AI readiness

When you open a new AI Search Specialist chat, an **Audit my website's AI readiness** chip appears as a conversation starter.

Clicking the chip sends the message and triggers an AEO Audit — an evaluation of your website's readiness for AI-powered search engines. Results appear inline in the chat.

### Audit cadence

The AEO Audit runs on demand, once per calendar month.

- If an audit has already run this month, the AI Search Specialist surfaces the most recent report with a relative run date (for example, "Last run 3 days ago").
- Your first AEO Audit runs automatically when you activate Local SEO Pro, so a report is available as soon as the initial run completes.

## Requirements

The AI Search Specialist is available with Local SEO Pro.
